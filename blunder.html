¿Qué es el Module Bundler y cómo funciona?

  • Datos destacables
  • Enlaces Externos
  • Resumen

Datos destacables

Los paquetes de módulos son herramientas que los desarrolladores frontales utilizan para agrupar módulos JavaScript en un solo archivo JavaScript que se puede ejecutar en el navegador. Ejemplos de paquetes de módulos modernos son: webpack, rollup, fusebox, parcel, etc.

Se requiere el agrupador de módulos porque:

  1. El navegador no es compatible con el sistema de módulos, aunque esto no es del todo cierto hoy en día.
  2. Le ayuda a administrar la relación de dependencia de su código, cargará módulos en orden de dependencia para usted.
  3. Le ayuda a cargar sus activos en orden de dependencia, activo de imagen, activo CSS, etc.

Resumen

Entonces, un resumen de lo que se debe tener en cuenta es:

  1. El paquete de módulos nos ayudó a combinar múltiples módulos JavaScript en 1 archivo JavaScript.
  2. Los diferentes paquetes de paquetes son diferentes, y hemos examinado 2 de los paquetes modernos, webpack y rollup
  3. La "forma de paquete web": utiliza el mapa del módulo, usa la función para envolver cada módulo, tiene un código de tiempo de ejecución que pega el módulo juntos.
  4. La "forma acumulada": paquete más plano y más pequeño, no usa la función para envolver el módulo, el orden importa, requiere una clasificación basada en la dependencia, la dependencia circular puede no funcionar.
